Hướng dẫn file câu hỏi

Ứng dụng có thể tự động nhận diện hoặc cho phép bạn tự ánh xạ cột. Một file chuẩn thường có các cột:

  • Nội dung câu hỏi: Cột chứa câu hỏi cần trả lời.
  • Các đáp án: Các cột chứa đáp án (ví dụ: Đáp án A, B, C, D...). Càng nhiều đáp án càng tốt.
  • Đáp án đúng: Cột chứa đáp án chính xác (chứa chữ cái A, B, C, D hoặc nội dung đáp án).
  • Giải thích (tùy chọn): Cột chứa giải thích tại sao đáp án đó đúng.

Chưa có sẵn file? Hãy tải file mẫu Excel để nhập câu hỏi của bạn hoặc trải nghiệm nhanh bằng dữ liệu mẫu:
